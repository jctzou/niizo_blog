手作教室

Amy的上課方式是讓學員隨選課程，自己挑自己喜歡的作品來學，所以，可能一個班六個人，都在上不同的作品，就不會有等待別人進度的問題。
也因為大家程度也不同，每個包款都依照難易度設計適合的上課堂數，考慮仍有些程度的差異，超過上課堂數也沒有關係，每個作品都還有一堂的補課可預約喔。

我們的包款都有內裡的設計，如何採用機縫的方式讓內裡都接合的漂亮，是布包製作上非常重要的精髓，往往是最後卡關的關鍵，雖然多了裡布製作需要花費更多的精神及難度，但對於布包學習者來說，是自我挑戰及技術提升非常重要的修練，歡迎您一起來玩。

上課時間：

  • 週二班  09:30~15:30(午休一小時，一堂課共五小時)
  • 週六班  13:00~18:00 (每個月兩次)

> 規劃新的課程模式中，目前暫停開課中

 

優惠方式

  • 兩人以上同報，學費九折。
  • 早鳥優惠：凡報名下個月份課程，並於本月完成繳費，學費九五折。

優惠方式

  • 兩人以上同報，學費九折。
  • 早鳥優惠：凡報名下個月份課程，並於本月完成繳費，學費九五折。

上課時數限制，報名前請詳讀

  1. 填寫報名表時，會請您填寫第一次預約上課的日期，Amy會在回覆信件時與您確認接下來的上課日，由於預約名額都是特定為您保留的，預約後若有異動，請在上課前三天告知，若是無故不到、遲到或早退，也算用了一堂課喔。
  2. 上課請務必準時， 就能在課程設計的時間內完成作品。
  3. 每個包款都依照難易度設計適合的上課堂數，考慮仍有些程度的差異，若仍無法順利完成，每個作品都還有一堂的補課可預約。若需再增加研習時數，以一小時200元計算。
  4. 從首次上課日起算，課程需於90天內完成，逾期將無法接受預約。

一些注意事項

  1. 隨選班採預約制，每個時段人數上限是6人，請事先預約及完成繳費以方便準備材料。
  2. 您可以帶自己的布來搭配，但課程原搭配的材料無法幫您替換或折抵。
  3. Amy的教學中，會指導版型與尺寸的關係，同學常因求好心切，有很多idea想實現出來，若因此需要改變做法或版型的結構，則無法協助。
  4. 布料裁切時請務必小心，若剪錯了需另外再購買。
  5. 工作室備有裁縫車供學員使用，但個人工具請自行準備。(例布剪、珠針、水消筆、粉土、手縫針，若有需要工作室也可幫您代購 )
